### Added
- Initial release
- Modern Python project template with uv, ruff, mypy, pytest
- Multi-platform Docker build targeting linux/amd64 and linux/arm64
- GitHub Actions CI, Docker/GHCR, CodeQL, and Release workflows
- VS Code Dev Container configuration
- Pre-commit hooks with ruff and mypy
- Makefile with common development commands
